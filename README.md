A simple Personal Finance Tracker web application built using HTML,CSS,JavaScript, PHP, MySQL, and Bootstrap.

Features:-
1.User Registration & login
2.Add Income and Expenses
3.Add Monthly Budget
4.Add Expenses
5.View Budget and Spent Amount
6.Dashboard with charts for visualization

Technologies Used:-
-HTML
-CSS
-JavaScript
-PHP
-MySQL
-Bootstrap

Setup to run the Project:-
1.Install Xampp
2.Start Apache and MySQL
3.Clone this repository
4.Import database into phpmyadmin
5.Open in browser
